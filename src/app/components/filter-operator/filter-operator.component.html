Filter

  • This operator will filter the values from source Observable based on the predicate function
  • The predicate function , will return a boolean value.
  • Output will get filtered if the function returns a truthy value.

Code:

            
                Observable$.pipe(
                    filter(person=> person.age < 40)
                )

Example:

Before filter
First Name Last Name Age Gender
{{person.firstName}} {{person.lastName}} {{person.age}} {{person.gender}}
After filter (age < 40)
First Name Last Name Age Gender
{{person.firstName}} {{person.lastName}} {{person.age}} {{person.gender}}
