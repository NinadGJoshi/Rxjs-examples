Reduce

  • In reduce operator, accumulator function is used on the input observable
  • Accumulator function will return the accumulated value in the form of an observable, with an optional seed value passed to the accumulator function.
  • It will return an observable that will have a single accumulated value.

Code:

            
                Observable$.pipe(
                    reduce((total, product: any) => total + product.price, 0)
                )
