Map

  • Runs a projection funtion for each emitted value by source observable
  • Used for transformation of the source values
  • Retuns observable of newly transformed values

Code:

            
                Observable$.pipe(
                map(
                    (person: any) => (
                      {{'{'}}
                        ...person,
                        fullName: `${{'{'}}person.gender === 'male'?'Mr.':'Ms.'{{'}'}}${{'{'}}person.firstName{{'}'}} ${{'{'}}person.lastName{{'}'}}`
                      {{'}'}}
                    )
                  ))

Example:

Before Map
First Name Last Name Age Gender
{{person.firstName}} {{person.lastName}} {{person.age}} {{person.gender}}
After Map
First Name Last Name Full Name Age Gender
{{person.firstName}} {{person.lastName}} {{person.fullName}} {{person.age}} {{person.gender}}
