ForkJoin

  • forkJoin is an operator that takes any number of input observables which can be passed either as an array or a dictionary of input observables.
  • It will wait for all passed observables to emit and complete and then it will emit an array or an object with last values from corresponding observables.
  • It will not emit more than once and it will complete after that.

Code:

            
                forkJoin([Observable1$, Observable2$,...........])
                .subscribe(([last_emitted_val1, last_emitted_val2, ...])=> {{'{'}}...{{'}'}})
